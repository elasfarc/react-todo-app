# Todo list

## Built With

- ReactJS
- CSS

## Getting Started

To get a local copy up and running follow these simple example steps.

### Setup

```bash
git clone https://github.com/elasfarc/react-todo-app.git
cd ./react-todo-app
```

### Install

```bash
npm install
```

### To Run it

```bash
npm start
```

### Build

```bash
npm run build
```
